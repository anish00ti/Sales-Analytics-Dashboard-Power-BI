# Sales-Analytics-Dashboard-Power-BI

This repository contains a multi-page interactive sales dashboard built using Power BI.

## Dashboard Highlights

- Built KPI cards to track high-level business performance.
- Added drill-through pages for deeper record-level analysis.
- Implemented slicers and bookmarks for guided and flexible navigation.
- Created DAX measures to support dynamic, real-time metric calculations.

## Analysis Coverage

- Sales and profit trends
- Customer behavior and segmentation
- Product performance insights
- Return rate tracking

## Interactive Filtering

The dashboard supports dynamic filtering by:

- Date
- Category
- Region
- Customer
